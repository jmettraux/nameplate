
= nameplate.rb

a twitter oriented nameplate generator (RubyKaigi-ish)

  $ git clone http://github.com/jmettraux/nameplate.git
  $ cd nameplate
  $ bundle install
  $ ruby nameplate.rb

go to

  http://127.0.0.1:4567/nameplate/jmettraux

basically

  http://127.0.0.1:4567/nameplate/{twitter-id}
